## this is chenfan's master dissertation project: Uncertainty Based Exploration for Deep Reinforcement Learning

### run the program (please read the main.py for arguments)
> python main.py 

### structure
agent: RL algroithm agents
env: environment for RL
imgs: saved images
log: saved log file
models_saved: saved models weights
networks: NNs in pytorch
runs: log file for tensorboard
toy_place: for toy testing
visualize: Transfer log to figures
buffer.py: RL buffer class 
logger.py: logger class 
main.py: main script
utilis.py: useful tools
